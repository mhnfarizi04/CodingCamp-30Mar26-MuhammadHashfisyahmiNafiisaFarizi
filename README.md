## ✨ Features
- **Real-Time Balance Tracker**: Automatically calculates total balance from income and expenses.
- **Smart Validation**: 
  - Prevents "Ghost Balance" (cannot add expense if balance is $0).
  - Deletion Guard (cannot delete income if it causes a negative balance).
- **Dynamic Visualization**: Interactive Pie/Doughnut chart powered by **Chart.js**.
- **Custom Categories**: Users can add their own custom spending categories.
- **Dark/Light Mode**: Features a smooth transition with a "Heartbeat" animated toggle button.
- **Persistent Storage**: All data is saved locally using the **Browser Local Storage API**.
- **Responsive Design**: Fully optimized for mobile and desktop views using **Bootstrap 5**.
